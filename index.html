fdx

fdx is a Golang package for working with XML files produced by many script writing programs (e.g. Trelby, Amazon Storywriter, Celtx and of course recent versions of Final Draft). This package also includes a demonstration command line program called fdx2txt which will read an fdx file and render plain text.
